# Development Scripts

## Overview
Development utility scripts for UniversalOS developers.

- `check-style.sh` - Check coding style
- `validate-structure.sh` - Validate directory structure
- `clean-build.sh` - Clean build artifacts
- `update-deps.sh` - Update dependencies

## Purpose
Development scripts help with:
- Environment setup
- Code formatting
- Style checking
- Structure validation
- Build cleanup
- Dependency management

## Usage
```bash
# Set up development environment
./scripts/dev/setup-dev-env.sh

# Format all code
./scripts/dev/format-code.sh

# Check coding style
./scripts/dev/check-style.sh

# Validate directory structure
./scripts/dev/validate-structure.sh

# Clean build artifacts
./scripts/dev/clean-build.sh
```

## Development Workflow
1. Set up environment: `setup-dev-env.sh`
2. Make code changes
3. Format code: `format-code.sh`
4. Check style: `check-style.sh`
5. Build and test
6. Commit changes

## Requirements
- Development tools installed
- Git repository initialized
- Write access to project files
