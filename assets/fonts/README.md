# Fonts

## Overview
Font assets for UniversalOS desktop environment and applications.

## Purpose
Provides fonts for:
- Desktop UI elements
- Terminal and code editors
- Document rendering
- System messages
- Application text

## Font Categories
- **UI Fonts**: Clean, readable fonts for interface
- **Monospace**: Fixed-width fonts for code and terminal
- **Serif**: Traditional fonts for documents
- **Sans-serif**: Modern fonts for UI

## Recommended Fonts
- **UI**: Noto Sans, Inter, Roboto
- **Code**: Fira Code, JetBrains Mono, Source Code Pro
- **Documents**: Noto Serif, Merriweather

## Font Formats
- **TTF**: TrueType Font
- **OTF**: OpenType Font
- **WOFF/WOFF2**: Web fonts (if needed)

## Installation
Fonts are automatically installed with UniversalOS and available system-wide.
