# Images

## Overview
Image assets for UniversalOS including logos, icons, and wallpapers.

## Structure
- `wallpapers/` - Desktop wallpapers

## Coming Soon
- `logo.png` - UniversalOS logo
- `icon.svg` - Application icon
- `splash.png` - Boot splash screen
- `screenshots/` - Application screenshots

## Purpose
Provides visual assets for:
- Desktop environment
- Boot splash screen
- Application icons
- Documentation
- Marketing materials

## Image Formats
- **PNG**: Raster images (logos, screenshots)
- **SVG**: Vector graphics (icons, scalable logos)
- **JPEG**: Photos and wallpapers

## Guidelines
- Use high-resolution images
- Provide multiple sizes when needed
- Use SVG for scalable graphics
- Optimize file sizes
- Include source files when possible

## Wallpapers
Desktop wallpapers in `wallpapers/`:
- Multiple resolutions (1920x1080, 2560x1440, 3840x2160)
- Light and dark variants
- Quantum-themed designs
