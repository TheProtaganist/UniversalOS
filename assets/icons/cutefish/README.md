# Cutefish Icon Theme

## Overview
Default icon theme for the Cutefish desktop environment in UniversalOS.

## Design Philosophy
- Modern and clean
- Consistent visual language
- Friendly and approachable
- Professional appearance
- Clear metaphors

## Style Guidelines
- Rounded corners
- Soft shadows
- Vibrant but not overwhelming colors
- Consistent stroke width
- Unified perspective

## Icon Categories

### Applications (apps/)
Icons for installed applications:
- File manager
- Terminal
- Text editor
- Settings
- Web browser
- And more...

### Actions (actions/)
User action icons:
- Copy, paste, cut
- Save, open, close
- Play, pause, stop
- Add, remove, edit
- And more...

### Devices (devices/)
Hardware device icons:
- Computer, laptop
- Phone, tablet
- Printer, scanner
- Storage devices
- And more...

### Places (places/)
Location and folder icons:
- Home folder
- Documents, downloads
- Desktop, pictures
- Network locations
- And more...

### Status (status/)
System status indicators:
- Network status
- Battery level
- Volume level
- Notifications
- And more...

### Mimetypes (mimetypes/)
File type icons:
- Text files
- Images
- Videos
- Audio files
- Archives
- And more...

## Sizes
All icons available in:
- 16x16, 24x24, 32x32, 48x48, 64x64, 128x128, 256x256
- Scalable (SVG)

## Color Palette
- Primary: #3498db (blue)
- Secondary: #2ecc71 (green)
- Accent: #e74c3c (red)
- Neutral: #95a5a6 (gray)
