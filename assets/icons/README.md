# Icons

## Overview
Icon themes for UniversalOS desktop environment.

## Structure
- `cutefish/` - Cutefish desktop icon theme
- `quantum/` - Quantum-themed icons

## Purpose
Provides icon themes for:
- Desktop applications
- File types
- System actions
- Status indicators
- Quantum features

## Icon Themes

### Cutefish
Default icon theme for Cutefish desktop:
- Modern, clean design
- Consistent style
- Multiple sizes
- Light and dark variants

### Quantum
Quantum-themed icons:
- Quantum computing features
- Scientific applications
- Quantum Control Center
- M-Wave storage

## Icon Sizes
Icons provided in multiple sizes:
- 16x16 (small UI elements)
- 24x24 (toolbar icons)
- 32x32 (standard icons)
- 48x48 (large icons)
- 64x64 (extra large)
- 128x128 (very large)
- 256x256 (high DPI)
- Scalable (SVG)

## Icon Categories
- **Applications**: Application launchers
- **Actions**: User actions (copy, paste, etc.)
- **Devices**: Hardware devices
- **Places**: Folders and locations
- **Status**: System status indicators
- **Mimetypes**: File type icons

## Format
- **SVG**: Scalable vector graphics (preferred)
- **PNG**: Raster images (multiple sizes)

## Guidelines
- Follow icon theme specification
- Consistent visual style
- Clear and recognizable
- Appropriate metaphors
- Accessible colors

## Related
- `src/desktop/` - Desktop environment
- `src/desktop/settings/` - Icon theme settings
