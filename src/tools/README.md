# Tools - System Utilities

## Overview
System utilities and tools for managing UniversalOS, including update, rollback, bootloader installation, and quantum state management.

## Structure
- `universalos-update/` - System update tool
- `universalos-rollback/` - Rollback tool
- `helix-boot-install/` - Bootloader installer
- `quantum-state/` - Quantum state manager
- `mwave-manager/` - M-Wave storage manager

## Purpose
Provides essential system management tools:
- System updates with atomic transactions
- Rollback to previous system states
- Bootloader installation and configuration
- Quantum state inspection and management
- M-Wave storage administration

## Key Features
- Atomic system updates
- Safe rollback mechanism
- Easy bootloader installation
- Quantum state visualization
- M-Wave storage tools

## Development
- Command-line tools
- Written in Rust and Python
- Integration with system services
- User-friendly interfaces

## Related Components
- `src/helix/` - Helix kernel
- `src/quantum/` - Quantum subsystems
- `src/bootloader/` - Helix Boot
